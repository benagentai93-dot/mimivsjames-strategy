📋 完整交易SOP流程

從選股、進場到出場的系統化操作流程。每筆交易都依循這套 SOP，用紀律取代情緒，用系統取代直覺。先問長線或短線？再決定週期與策略。

1

選股與定位

先問自己：長線 or 短線？

• 長線重「質」(護城河、估值合理性)

• 短線重「勢」(動能、量價、催化劑)

Alpha 選股條件：

• Beta 合理 + Alpha 表現突出

• 無事件驅動下的量價上漲

• 均線健康：5>10>20 多頭排列

• 日成交量 ≥ 5日均量 1~1.5倍

時間週期選擇：
波段(1~4週)：日K為主，60分K為輔
短線(1~7日)：60分K為主，15分K為輔
2

大週期找趨勢

日線/週線確認方向：

• 均線糾結後突破 + 三線張口放量

• 20MA 斜率連續 3-5 根向上

• VIX 高點持續下降（恐慌退潮）

趨勢啟動確認：

• SPY 突破箱體 3 日不回跌

• 5MA/10MA/20MA 糾結後張開上彎

• 量能不失衡 (1~1.5倍均量)

21MA 動能判斷：
日內低點在 21MA 之上 = 強勢動能
跌破 21MA = 動能鈍化，攻守切換
3

量穩結構確認

站穩支撐 + 回檔量縮：

• 收盤站穩均線

• 拉回量縮再放量上攻

• K線下引線長 = 轉強訊號

VWAP 盤中確認：

• 開盤30分後觀察站上/跌破

• 多次回測不破 = 強勢整理

• 下跌量縮 + 上攻放量 = 進攻

真趨勢 vs 假突破：
真：斜率增、線拉開、量放大、多日站穩
假：角度平、線糾結、量縮、單日即回
4

進場 (右側交易)

進場條件 (至少符合2項)：

• 收盤站穩均線 (20MA/21MA)

• 放量突破箱體上緣

• 拉回量縮再放量上攻

• K線下引線長，轉強訊號

盈虧比檢查：

• R:R ≥ 2 才能進場

• 日線找不到 R:R>2 就空手

進階進場法：
日線確認方向 → 60分K等洗盤 →
站回8EMA進場，停損設假跌破低點
5

停損設定

停損方法 (擇一)：

• 關鍵支撐下方 1~2%

• ATR × 0.5~1.5 倍

• 前一波低點下方一點

• 1% 救命公式 (總資金 × 1%)

執行原則：

• 收盤跌破均線才算，盤中刺穿不算

• 箱型整理用箱底低點，不用均線

• 搭配 MACD/KD 死叉加強確認

風控紀律：
單筆虧損 ≤ 總資金 1~2%
日虧 ≤ 3~5% 強制關機
同方向總曝險 < 5%
6

停利與加碼

分批停利 SOP：

① 達目標價賣 30%，停損移至成本價

② 調高目標再賣 30%，停損移至第一目標

③ 剩 40% 不設目標，移動止盈單

加碼條件（反馬丁）：

• 已獲利 + 趨勢延續

• 回踩支撐量縮價穩再放量

• 每次加碼後停損上移

停利觸發訊號：
爆量下跌長黑K、突破短期生命線(8/21EMA)、
VRVP前波高套牢區壓力

機構投資思維與操作紀律筆記

投資心理學的終點，不是靠意志力對抗人性，而是設計一套「自動導航系統」。結合右側加碼、嚴格停損與再平衡，讓你在市場恐慌或狂熱時，都能依循SOP做出正確決策。

📊

投資系統五大模組

  1. 目標模組 (定義IRR與回撤)
  2. 配置模組 (核心70%-衛星30%架構)
  3. 估值模組 (P/E分位與盈利品質濾網)
  4. 再平衡模組 (紀律性觸發買賣)
  5. 檢視模組 (定期回顧與剔除)
▣

1% 救命公式與資金控管

單筆交易的最大虧損，絕對不能超過總資金的 1%~2%。進場前先算好「停損價」，用 (總資金×1%) / (買價-停損價) 來決定你可以買幾股。這保證你連錯10次都不會畢業。

📈

右側反馬丁金字塔加倉

永遠不要「左側攤平」(越跌越買)。加倉必須在「已經獲利」且「趨勢延續」時進行。資金拆分為 50% → 30% → 20%。每加碼一次，就把停損點往上移，保護已實現利潤。

🛡

再平衡與防禦模式

設定目標權重(如10%)與容忍區間(±2%)。漲超就賣，跌深就補，克服處置效應。

當總資產回撤達 -15% 時啟動防禦：凍結主題股加碼，資金僅補核心ETF，產業上限下調至30%。

⚠

假突破 (2B法則) 與破底翻

開盤急拉別急著追。突破必須帶量 (1.2倍以上)，且回測不破箱體。如果突破後「量縮上行留長上影線」，或跌回箱體內，就是主力的假突破誘多，應立即停損出場。

🎯

對沖 (Hedging) 與最後一擊

對沖不是放空，是買保險。用總資金 1-2% 買 Put 防禦宏觀大跌。

反馬丁最後的 10-20% 加倉，可使用 Long Call Spread (買權多頭價差)，風險有限且能參與主升段爆發。

📈

均線實戰應用 (5/10/20/50/200)

5MA短線動能，跌破就跑 10MA波段節奏線，跌破代表動能轉弱 20MA波段防守線，跌破後要減碼 50MA跌破三日收不回 = 資金撤離訊號 200MA年線，空轉多最重要的確認線
判讀四要素：方向(收盤在均線上方)、角度(斜率=力道)、間距(三線拉開=趨勢一致)、次序(5>10>20=多頭排列)
📊

量價關係判讀法則

價漲量增有資金真正進場，趨勢健康 價漲量縮散戶追價、主力觀望，假多頭徵兆 價跌量縮通常是洗盤，主力仍在場 價跌量增主力撤出，趨勢反轉

突破確認：成交量放大至近 5 日平均的 1~1.5 倍以上，收盤價站穩壓力區上方。無事件驅動下的量價上漲，參考價值更高。

📊

布林帶 (Bollinger Bands) 應用

盤整盤：價格跑到 +2σ 代表偏離平均，均值回歸機率高。

趨勢盤：貼著上軌走 + 回檔量縮 + 5MA不破 = 趨勢尚未耗竭，可持有。

風險訊號：頻繁跌破5MA且3日站不回、價格回不到上軌向中軸靠近、中軸走平或下彎。一旦跌破中軸(20MA)，股價往下跳一個級距(如50MA)。
📊

VWAP 主力成本線判讀

開盤30分後用5/15分K觀察：

穩站VWAP偏多盤，回測不破再進場 拉高跌破假拉真出，轉弱訊號勿追 上下洗盤無方向盤，觀望 多次不破強勢整理，下跌量縮+上攻放量=進攻

無量突破VWAP可能為假突破，需放量且重新站回確認。

📈

止跌與見底訊號判讀

止跌訊號（可小量試單）：

三天不創新低、長陽K創近期新高且量增、走勢橫盤不再下滑、下影線變長、利空不跌。

見底訊號（右側進場/加碼）：

突破壓力線、站回重要均線(20/60/200MA)、出現長紅K創小新高。空轉多最重要：站上200MA年線。

輔助確認：MACD柱縮短即將金叉、RSI回升>40、底部愈來愈高、利空不跌反漲。

情緒斷路器（連虧降速機制）

防止連續虧損後產生「把錢贏回來」的報復性交易，必須設定並強制執行自動煞車：

⚠ 日風控（每日最大虧損）
當日虧損達帳戶總資金 3% → 強制關機、離開房間
（虧3%容易賺回；虧10%心態崩潰想賭大）
🔴 連輸 2 筆 → 下一筆部位強制減半
代表今天節奏與市場不合，降速保護本金
⛔ 連輸 3 筆 → 直接強制關機（Stop Trading）
紅燈亮起，去睡覺或運動，絕對禁止再交易
核心觀念：空手也是倉位，讓你保有下次進場的權利。

每週五抓鬼儀式

每週五收盤後（或每月一次），花 30 分鐘打開交易日誌，抓出重複出現的虧損行為模式。

抓鬼流程：
• 翻閱本週所有「紅筆圈起」的錯誤交易
• 找出情緒盲點，例如：「70%的虧損都發生在開盤前10分鐘去追高(FOMO)」
• 或是「只要心情煩躁時下單，勝率幾乎是零」
設定「一條天條」：
針對本週最常讓你賠錢的壞習慣，設定一條新規則嚴格執行。
例如：「沒有回踩，死都不追」

完整復盤 SOP（4步驟）

將交易串聯成「觀察 → 進場 → 出場 → 檢討」的 SOP，藉此不斷進步：

Step 1：填寫交易日誌
交易結束後，確實記錄進出場點位、劇本、觸發條件與當下情緒
Step 2：決策四象限歸類
客觀評估這筆是靠實力還是運氣，有沒有因情緒提早出場或衝動加碼
Step 3：檢視紀律執行度
回頭檢查是否按照設定條件進場？有沒有嚴格設好停損？
Step 4：抓鬼與系統優化
透過週期性檢討（每週五），抓出盲點與情緒漏洞，轉化為下一週的「天條」

進階操作選股 (風險低、報酬高、續航力強)

均線共振 + 量價配合的三階段建倉法，在趨勢初期介入、趨勢確認後加碼、回踩支撐時追加，用移動止盈鎖住獲利。

① 試單建倉 (10-20%)
進場訊號：
• 5MA 剛上穿 10MA / 20MA
• 20MA 開始下彎變平 / 微微上彎
• 成交量開始放大，但還沒爆天量
• 股價站上所有均線
停損：跌破箱體 或 10MA
② 趨勢確認加碼 (30-40%)
加碼條件：
• 均線多頭排列確認
• 5MA > 10MA > 20MA
• 量價齊揚，趨勢延續
• 已處於獲利狀態
停損：跌破箱體上緣 或 20MA 下方一點
③ 回踩追加 (每次 10-20%)
追加時機：
• 股價踩回箱體上緣 或 20MA
• 等量縮價穩後再放量上攻
• 確認支撐有效再進場
• 每次加碼後上移停損
移動止盈：用前一波回踩低點做防守
💡
核心邏輯： 在均線共振的初期用小倉位試單（低風險），趨勢確認後才大幅加碼（高報酬），回踩支撐時追加並用移動止盈保護利潤（續航力強）。整個過程嚴格遵守右側交易紀律，絕不左側攤平。
此工具基於機構再平衡、1%風控公式與反馬丁策略設計。
核心概念：70% 核心配置 + 30% 動能衛星部位 | 右側交易 | 嚴格停損

⚔ 盈虧比與分批停利計算機

為什麼你總是「賺小賠大」？因為你沒有在進場前算好盈虧比。機構標準：盈虧比 (R:R) > 2 才能進場。先設好停利目標與停損防守價，讓系統幫你判斷這筆交易值不值得做。

📈 財報後 AVWAP 應用 (Anchored VWAP)

📈 AVWAP 怎麼計算？

公式：

典型價 TP = (High + Low + Close) / 3
AVWAP = Σ(TPᵢ × Volᵢ) / Σ(Volᵢ)
其中 i 從財報日後第一天開始累積

與日內 VWAP 不同，AVWAP 跨越多日累積，是機構從財報後持續建倉的「真實平均成本」。

🔌 財報後 AVWAP 帶寬（±1σ）

進階版 AVWAP 可以加上「標準差帶寬」：

+1σ機構高點出貨區，分批停利 AVWAP機構成本線，中性區/觀察位 -1σ超跌修復區，但需謹慎

系統計算的「偏差率」即為股價相對 AVWAP 的位置，超過 ±5% 代表趨勢明顯。

⚠ 財報後操作風險注意
跳空缺口回補：財報跳漲後，回補缺口前先別重押
IV 崩潰：財報後 IV 急跌，期權持有者需警覺
量能驗證：財報後放量站上 AVWAP 才算有效突破
連續涉及財報：AVWAP 只在財報後 1-4 週最有參考性
Anchored VWAP 是機構追蹤「事件後資金成本」的核心工具。財報後使用 AVWAP 判斷多空分界，配合量能驗證，是吃到財報後「第二波段」最可靠的方法。

型態操作指南 — 美股機構策略系統

四大市場型態完整辨識邏輯與實戰 SOP。止跌不等於見底，滯漲不等於轉空，突破不等於真突破。每一個型態都有明確的確認條件、進場點、止損線與獲利目標。

📈

底部型態 — 核心定義

底部型態是市場從劇烈下跌轉為主力完成吸籌、發出轉強訊號的完整結構。止跌 ≠ 見底：止跌只是拋壓減輕的過渡期；真正的底部是套牢籌碼沉澱、主力接籌後發出確認突破訊號。重點不在尋找最低點，而是等待「確認轉強」的右側進場訊號。

🔎

前期觀察：7大止跌訊號

出現 1～2 個即代表拋壓減輕

  1. 三天不創新低 — 股價連續幾天不再破底
  2. 長陽K線拔起 — 大跌後首次出現明顯長紅K
  3. 走勢轉橫盤震盪 — 從溜滑梯下跌轉為左右震盪
  4. 低檔量增不跌 — 成交量增多但價格未跌破
  5. 支撐處反彈加快 — 跌到特定位置買盤迅速出手
  6. 下影線變長 — 盤中下殺但收盤被拉回
  7. 利空測底不跌 — 壞消息出現股價反而撐住（利空出盡）
✅

右側確認：底部三部曲

三個條件缺一不可，確認後才是真底部

① 突破下降趨勢線
股價向上穿透壓制的下坡線，空頭失去主導權
② 重新站上關鍵均線
收復 20MA / 60MA 或 200MA 年線並能撐住
③ 放量長紅K創短期新高
帶大成交量實體紅K突破震盪區間，市場信心真實回籠
📊

特殊底部結構

  • 底底高、頭頭高 — 每次回檔低點墊高，漲有力跌無力
  • 破底翻（2B做多法） — 主力砸穿前低爆大量後迅速拉回，RSI/MACD底背離
  • IPO底部 — 新股震盪清洗後的杯形/碗形結構，量縮時跌漲時放
  • 籌碼鐵底(VRVP) — 底部落在 POC/HVN 節點，機構建倉大本營
👉

進場策略 Entry

右側確認進場
等底部三部曲訊號出現後，等待回踩量縮不破再買入，勝率最高。
最佳進場時機
第一次「量縮回踩 → 再度放量上攻」時，是最穩健且確認度最高的買點。
🛡

止損 & 目標

止損設定
設在突破 K 線最低點下方，或回踩支撐節點（POC/HVN）下方 + 0.5～1倍 ATR 緩衝
目標設定
前波高點 / VRVP 阻力區 HVN。達第一目標後分批減倉，剩餘部位用 10MA/21EMA 移動防守
⏳

滯漲型態 — 核心定義

股價上漲一段時間後到達相對高點，無法繼續創新高，轉為水平橫向整理。滯漲 ≠ 轉空：可能是主力短暫休息等待題材，或是鎖碼護盤（中小型股常見）。必須視為「黃色警戒」，雙手準備：帶量突破箱頂 → 繼續做多；跌破箱底 → 確認頂部，離場。

🔎

辨識訊號

K線特徵
連續 3 個交易日以上無法突破前高；K 線頻繁出現上影線、十字線、流星線；高點越來越低（Divergence）
技術指標轉弱
RSI 從高檔鈍化後下彎（從 70/80 回落）；MACD 死叉或柱狀體縮小；K 棒跌破 10MA 且無法快速站回
量價背離 / 籌碼異常
高檔量增價不漲（主力在賣散戶在接）；暗池資金流入急降；散戶大量買 OTM Call 但股價橫盤不動
⚖

操作 SOP

防守：降倉降槓桿
倉位比重降至 70% 以下；K 線出現轉弱（黑K/長上影/跌破 5MA），短線分批止盈；長線掛好對沖（買 Put）
止損：預設防守線
在前一波低點或箱型下緣設定價格警報；跌破頸線/20MA → 確認頂部，必須果斷離場
進場：等待右側突破
穩健型：等帶量突破箱頂 + 量縮回踩不破再進場
激進型：箱底支撐試單，停損設在箱底下方，目標看回箱頂
⚡

突破型態 — 核心定義

股價在盤整或面臨關鍵壓力時向上穿透阻力，代表多方取得壓倒性優勢，準備開啟新一段上漲趨勢。市場充斥主力用來誘多的「假突破」，真突破必須同時滿足三大核心條件。

✅

真突破三大必要條件

① 實體K線收盤站上壓力位
盤中穿過不算，必須「實體」明確收在壓力位之上（收盤價 = 市場共識）
② 成交量明顯放大（推升量）
突破當天成交量 ≥ 過去 20～30 天均量的 1.5～2 倍，有真金白銀的新資金追買
③ 回測有撐 + 三日法則
放量上漲 → 量縮回踩不破 → 再放量過關；股價連續 2～3 天收在壓力位之上才確立
⚠

假突破警示（致命陷阱）

假突破（多頭陷阱）特徵
突破瞬間爆大量（是停損量非推升量），股價衝高後立刻卡住，1～2 根 K 線直接砸回盤整區間內
頂部 2B 法則（動能衰竭）
股價創新高（High 2）但站不穩，極短時間內跌回前一波高點（High 1）之下 → 多方力竭，準備反轉向空
輔助工具
達瓦斯箱型（Darvas Box）：帶量突破箱頂 = 明確看漲進場訊號
👉

進場策略 Entry

拒絕無腦追高。最佳右側進場點 = 突破後第一次量縮回測支撐不破，再度放量上攻時買入。避開假突破風險，勝率最高。

🛡

止損設定 Stop-Loss

停損設在突破 K 線最低點下方，或箱體上緣（新支撐）下方一點點。收盤跌回箱型整理區間內 = 突破失敗，果斷停損，不硬凹。

📈

目標 & 移動止盈

突破成功後進入主升段，不預設天花板。改用移動止盈：沿 5MA / 10MA 防守，每天收盤守在均線上就抱住，帶量跌破均線才分批獲利了結。

🔴

頂部型態 — 核心定義

股價在上漲一段時間後到達高點，無法繼續創新高並轉而反轉下跌的結構。圖形如「山」形，代表買盤力道無法延續、資金逐漸撤出，是趨勢由多頭轉空頭的關鍵轉折。通常在頂部形成前，會先出現一段「滯漲型態」。看到頂部確立，操作核心只有一個字：跑。

🔎

四大頂部圖形

M 頭型態（雙頂）
高檔出現兩次高點，第二次上攻無法突破前高，隨後跌破兩峰間的頸線支撐，圖形像「M」
頭肩頂型態
左肩 → 頭部（最高點）→ 右肩（低於頭部）→ 跌破頸線。中長期轉空最具參考價值的型態
箱型整理後破底
高檔失去動能進入橫向震盪，長時間無法突破箱頂，最終向下跌破箱底 → 頂部觸發
頂部 2B 法則（假突破/多頭陷阱）
短暫突破前高吸引散戶追買，1～2 根K線後迅速跌回前高之下。機構倒貨的強烈轉折訊號
✎

確認條件 & 操作 SOP

確認條件（兩個必要）
① 高點降低，反彈無法突破前高
② 跌破關鍵支撐（頸線或前波低點）
提前防守
疑似頂部（右肩形成中/高檔滯漲）→ 降槓桿，倉位降至 70% 以下；長線持股掛對沖保護（買 Put）
嚴格停損（最重要）
支撐線下方設價格警報，跌破頸線/關鍵點位 → 果斷停損或清倉，絕不凹單等解套
激進者：反手做空
2B假突破確認後進場做空，停損設在假突破最高點 + 1倍 ATR，盈虧比極佳
離場後策略
出場後耐心等待，沒出現明確底部訊號 / 均線轉強前，絕不急著買回

五大進階工具 — 散戶看不到，華爾街每天在用

VRVP + AVWAP + DIX + ADL + VVIX。五個工具組成完整的抄底保命系統：先找機構成本底部（VRVP × AVWAP 共振）→ 再確認機構真實意圖（DIX × ADL）→ 最後判斷市場恐慌是否見頂（VVIX）。

📈

① VRVP — 成交量分布圖（籌碼地圖）

傳統成交量告訴你「哪天」熱絡；VRVP 把成交量轉 90° 顯示「哪個價位」有大量真金白銀。

POC 控制點
圖表右側最長橫條 = 單一價位成交量最大 = 市場絕對共識價。機構大資金建倉防守的終極鐵底，最強支撐。
HVN 高成交量節點
橫條明顯偏長的籌碼山丘 = 大戶護盤 / 散戶攤平的聚集地，容易形成支撐反彈（次級防禦陣地）。
LVN 低成交量節點（真空區）
橫條極短的峽谷 = 籌碼真空，股價跌入後缺乏買盤，瞬間滑價加速下跌，直到碰到下個 HVN 才停。
VA 價值區域 — VAH / VAL
70% 總成交量集中的區間。VAH = 價值區高點；VAL = 價值區低點。這兩條線框出主力的「核心控盤範圍」。
▸ 股價帶量跌破 VAL → 趨勢由多轉空，若下方接 LVN 快速暴跌
▸ 股價突破 VAH 後回測不破 → 天花板成地板，高勝率起漲點
選擇權搭配策略
底部 POC/HVN 區：Sell Put / Sell Put Spread 收租（不跌破就收保費，跌破則以低成本接股）
跌入 LVN 真空區：買 Put Spread（買近價 Put + 賣下方 HVN 處 Put），降低避險成本又吃滿滑落價差
POC 成壓力後：不斷 Sell Call / Covered Call 降低被套持倉成本
⚓

② AVWAP — 錨定加權均價（機構 GPS）

一般均線計算時間與收盤價；AVWAP 將起點「錨定」在重大事件上，算出帶動趨勢的大資金真實平均成本。

常用錨點（重大事件）
  • 財報日（爆出天量/大跳空）
  • 聯準會降息/升息當天
  • 大盤連續暴跌後爆量長下影線（市場大底）
  • 重大政策/事件發布當天
判讀訊號
▲ 股價在 AVWAP 之上 = 主力都在獲利，多頭控盤，續抱
▼ 帶量跌破 AVWAP = 連主力都虧損/跑路，果斷停損
回測 AVWAP 量縮不破 = 最佳加碼位
標準差通道（2σ）
AVWAP ± 2σ = 95% 統計機率的極限邊界（極度過熱或超跌）。碰到 +2σ 上軌 = 短線獲利了結；碰到 -2σ 下軌 = 恐慌超跌的抄底參考線。
⭐ AVWAP × VRVP 共振買點（最高勝率）： 當 AVWAP（動態主力成本）與 VRVP 的 POC/HVN（靜態籌碼鐵底）指向同一個價位時，代表過去與現在的大資金成本都在此集結 = 防守成本最低、勝率最高的「共振鐵底區」。
👁

③ DIX — 暗池指數

機構動態監視器。機構大戶買賣不會在一般看盤軟體顯示，DIX 讓你看到他們在暗池裡的真實動作。

DIX 下降 / 低位
大盤暴跌時機構也在暗池跟著砸盤 → 反彈可能是假的，不宜接刀
DIX 上升 / 高位
大盤下跌但機構在暗池偷偷吃貨（吸籌）→ 下跌可能是誘空，真底部訊號
搭配 ADL 使用
DIX（機構意圖）× ADL（市場廣度）雙重確認，才能識破機構真正的劇本
📊

④ ADL — 騰落線（市場廣度透視鏡）

S&P500 是市值加權，容易被少數科技巨頭綁架。ADL 追蹤全市場真實上漲/下跌家數，看穿指數的真相。

指數漲但 ADL 不漲（背離）
機構用少數權重股拉高指數製造假象，廣大成分股其實在跌 → 反彈不健康，謹慎追高
指數跌但 ADL 不跌（背離）
機構用權重股壓盤製造恐慌，但廣泛成分股在吸籌 → 下跌是假的，底部正在形成
搭配 ATR 預警
ATR 增加 + ADL 廣度減弱 = 市場噪音大，下行風險升溫，應降倉
🔥

⑤ VVIX — 恐慌加速度

VIX 是恐慌程度；VVIX 是恐慌的「加速度」。精準抓出市場情緒崩潰的最高點。

VVIX 急速飆升
恐慌加速中，市場仍在崩潰進行式 → 等待、防守，不急著抄底
VVIX 高位後開始下彎
恐慌加速度見頂放緩 → 歷史上幾乎每次都是大盤反轉前 1～3 個交易日的預兆，進攻訊號
VVIX 低位平靜
市場情緒穩定 → 正常操作，可維持正常倉位
🔧

五工具整合 SOP — 抄底保命三步驟

1
找出機構成本底部區
用 VRVP / FRVP 找到資金密集價位（POC / HVN）
再用 AVWAP 找出機構動態持倉成本
▶ 兩者指向同一位置 = 共振鐵底，機構成本的底部區，最高勝率進場點
2
確認機構是買還是賣
用 DIX 判斷機構在暗池的真實動作（吃貨 or 砸盤）
用 ADL 看大盤指數的真相（廣度健康 or 僅靠權重股撐）
▶ DIX 上升 + ADL 不跌 = 反彈為真，可進場；兩者背離 = 反彈為假，等待
3
判斷恐慌是否見頂
用 VVIX 判斷市場情緒崩潰是在加速還是在減速
VVIX 高位下彎 = 恐慌最高點，通常是反轉前 1～3 日
▶ 三個訊號同時出現 = 「等待 → 進攻」的黃金切換時刻
DIX ADL VVIX VRVP + AVWAP 建議操作
↑ 吃貨 ↑ 廣度健康 ↓ 恐慌退燒 POC / HVN 共振支撐 積極進場，分批建倉
↑ 吃貨 → 混訊 → 高位震盪 接近支撐區 小量試單，嚴設停損
↓ 砸盤 ↓ 廣度轉弱 ↑ 恐慌加速 跌破 VAL 進入 LVN 等待 / 對沖 / 空手保命
↓ 砸盤 → 混訊 ↓ 開始下彎 接近 HVN / 下方有 HVN 觀察等待，備好彈藥
